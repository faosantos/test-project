Rodar a Seeder:
Uma vez que criou a classe de Seeder, e registrou o próximo passo é simplesmente rodar e criar o novo registro, em nosso caso criar um novo usuário, para isso rode este comando:

1
php artisan db:seed
Este comando vai rodar todas as classes de Seeders registradas na classe DatabaseSeeder.

É possível também especificar qual classe de seeder vai rodar ao executar o comando, veja:

1
php artisan db:seed --class=UsersTableSeeder
Neste caso mesmo que tenha várias classes de Seeders registradas vai rodar apenas a UsersTableSeeder.

Também é possível criar as Seeders no momento que criar as tabelas através do recurso de migration, veja como:

1
php artisan migrate:refresh --seed
Este comando vai recriar as tabelas e automaticamente rodas as Seeders registradas.
